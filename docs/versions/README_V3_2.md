# README V3.2

## Objet

`V3.2` est une version stabilisée du dictionnaire français minimal orientée travail, administration, argent et droit courant à `650` mots.

Elle étend `V3.1` sans le modifier ni le remplacer.

## Portée exacte

V3.2 étend V3.1 vers :

- travail ;
- administration ;
- argent ;
- droit courant.

Plus précisément, cette version permet de parler plus facilement de :

- emploi, métier, entreprise, bureau, collègue, responsable, client, projet, contrat, salaire, congé, horaire ;
- document, dossier, formulaire, demande, service public, mairie, numéro, adresse, identité, carte, preuve, signature, rendez-vous ;
- banque, compte, paiement, facture, dette, revenu, dépense, impôt, taxe, assurance, aide ;
- accord, obligation, autorisation, interdit, permis, responsabilité, plainte, juge, tribunal.

## Ce que V3.2 ne cherche pas à faire

- couvrir tout le droit spécialisé ;
- remplacer un vocabulaire administratif complet ;
- couvrir toute l économie ou le monde du travail fin ;
- fournir un langage juridique technique ;
- couvrir toutes les situations de police, de justice pénale ou de contentieux détaillé.

## Validation

La version stabilisée conserve les propriétés suivantes :

- liste fermée de 650 mots ;
- dictionnaire fermé sur cette liste ;
- table explicite de formes fléchies ;
- validation stricte sans mot hors vocabulaire ;
- 100 tests d expression travail, administration, argent et droit valides.

## Interprétation

V3.2 doit être compris comme une branche pratique stable pour les rapports ordinaires au travail, aux démarches, au paiement et aux règles courantes.

Une V3.3 deviendra pertinente si l objectif est d ajouter des notions comme `avocat`, `police`, `budget`, `loyer`, `chômage`, `procédure` ou `délai`.
