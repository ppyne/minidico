## Portée exacte

- Droit pratique
- Logement et budget
- Santé fine
- Santé mentale
- Travail social

## Ce que cette version ne cherche pas à faire

- couvrir tous les registres spécialisés ;
- remplacer une langue naturelle complète ;
- résoudre toutes les polysémies du français courant ;
- supprimer entièrement les définitions pivots ou quasi primitives ;

## Interprétation

V3_3 doit être compris comme une branche stable du corpus confortable, centrée sur Droit pratique, Logement et budget, Santé fine, Santé mentale.
