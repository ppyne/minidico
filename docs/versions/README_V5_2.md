# README V5.2

## Portée

V5.2 est une stabilisation sémantique du corpus confortable.

- la liste reste fixée à `1500` mots ;
- aucun mot n’est ajouté ;
- aucun mot n’est supprimé ;
- la wordlist V5 est conservée à l’identique dans [wordlist_v5_2.txt](/Users/avialle/dev/minidico/wordlist_v5_2.txt).

## Ce Que V5.2 Améliore

V5.2 améliore surtout les définitions faibles héritées de V5 et de V5.1.

- réduction forte des définitions de type `chose de ...` ;
- réduction forte des définitions de type `chose pour ...` ;
- remplacement de nombreux hyperonymes seuls ;
- amélioration de nombreux mots de transport, science, art, relations, maison et numérique ;
- conservation de la fermeture lexicale du système.

## Statut Lexical

V5.2 reste lexicalement fermée.

- `1500/1500` définitions conformes ;
- `0` mot hors vocabulaire ;
- `0` circularité directe simple.

## Position Dans La Série

V5.2 n’est pas une extension conceptuelle.

C’est une stabilisation sémantique de la même base lexicale à `1500` mots, destinée à rendre les définitions plus utiles, plus lisibles et moins dépendantes de placeholders pauvres.
