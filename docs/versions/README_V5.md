## Portée exacte

- Communication moderne
- Consolidation
- Qualités
- Relations sociales
- Vie domestique
- Émotions

## Ce que cette version ne cherche pas à faire

- couvrir tous les registres spécialisés ;
- remplacer une langue naturelle complète ;
- résoudre toutes les polysémies du français courant ;
- supprimer entièrement les définitions pivots ou quasi primitives ;

## Interprétation

V5 doit être compris comme une branche stable du corpus confortable, centrée sur Communication moderne, Consolidation, Qualités, Relations sociales.
