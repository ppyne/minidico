# README V3.1

## Objet

`V3.1` est une version stabilisée du dictionnaire français minimal orientée santé pratique à `600` mots.

Elle étend `V3.0` sans le modifier ni le remplacer.

## Portée exacte

V3.1 étend V3.0 vers :

- infection ;
- prévention ;
- examens ;
- pharmacie ;
- santé reproductive concrète ;
- urgence et conscience.

Plus précisément, cette version permet de parler plus facilement de :

- infection, virus, bactérie, allergie, contamination ;
- consultation, patient, test, analyse, radio, ordonnance, dose, pharmacie ;
- vaccin, vaccination minimale, hygiène et prévention ;
- organes et anatomie plus précis : poumon, estomac, rein, utérus, vagin, pénis, ovaire, testicule ;
- règles, contraception, préservatif, sexualité, fertilité, accouchement ;
- conscience, inconscience, saignement, signes de gravité plus concrets.

## Ce que V3.1 ne cherche pas à faire

- couvrir toute la médecine spécialisée ;
- remplacer un vocabulaire médical complet ;
- fournir un langage clinique fin pour toutes les pathologies ;
- couvrir toute la santé mentale détaillée ;
- couvrir les mesures vitales et l’accidentologie complète.

## Validation

La version stabilisée conserve les propriétés suivantes :

- liste fermée de 600 mots ;
- dictionnaire fermé sur cette liste ;
- table explicite de formes fléchies ;
- validation stricte sans mot hors vocabulaire ;
- 100 tests d’expression santé valides.

## Interprétation

V3.1 doit être compris comme une branche santé pratique stable : plus concrète que V3.0, tout en restant dans un français minimal non technique.

Une V3.2 deviendra pertinente si l’objectif est d’ajouter des notions comme `température`, `tension`, `accident`, `surdose`, `dépression`, `anxiété` ou des examens et traitements plus précis.
