# Release Notes V6.2

## Résumé

V6.2 est la version stable éditoriale et conceptuelle du corpus français contrôlé.

Cette publication fixe une base de référence à `1573` mots, lexicalement fermée, avec une couche morphologique nettoyée et une validation conceptuelle complète.

## Points principaux

- base stable : `1573` mots, `1573` entrées, `1227` formes ;
- héritage direct de V6.0 stable ;
- ajout de `12` concepts transversaux ;
- correction des accords suspects signalés en V6.1 ;
- nettoyage des formes orthographiquement fautives ciblées ;
- aucun nouveau domaine concret ;
- validation stricte réussie ;
- `300/300` tests conceptuels valides.

## Résultats de validation

- définitions conformes : `1573/1573` ;
- mots hors vocabulaire : `0` ;
- circularités directes simples : `0` ;
- phrases conceptuelles naturelles : `140` ;
- phrases conceptuelles acceptables : `160` ;
- phrases conceptuelles lourdes : `0` ;
- phrases conceptuelles impossibles : `0`.
