# Diff du rapport final V3.0

## Portée

Ce diff concerne uniquement le fichier `validation_report_v3_0_final.md`.

Aucun fichier lexical n a été modifié :

- `wordlist_v3_0.txt`
- `dictionary_v3_0.json`
- `forms_v3_0.txt`

## Changement appliqué

### Ancienne formulation

`La prochaine iteration doit surtout enrichir la precision des definitions, pas agrandir la liste.`

### Nouvelle formulation

`La prochaine iteration peut suivre deux axes : améliorer la précision de certaines définitions existantes, ou poursuivre l’enrichissement contrôlé par domaine, notamment en santé pratique, prévention, examens, infection et santé reproductive.`

## Effet

- suppression d une recommandation trop restrictive ;
- ouverture explicite à une poursuite contrôlée de l extension lexicale par domaine ;
- cohérence accrue avec la stabilisation V3.0 orientée santé.
