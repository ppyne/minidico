# Changelog V3.2

## V3.2 stable

Cette version fige la branche travail, administration, argent et droit courant sur une base de 650 mots.

### Base de départ

- point de départ : V3.1 stable à 600 mots ;
- conservation complète de la branche V3.1 ;
- ajout de 50 mots orientés travail, administration, argent et droit courant.

### Fichiers stabilisés

- `wordlist_v3_2.txt`
- `dictionary_v3_2.json`
- `forms_v3_2.txt`

### Couverture ajoutée

- travail : `emploi`, `métier`, `entreprise`, `bureau`, `collègue`, `responsable`, `chef`, `client`, `projet`, `tâche`, `réunion`, `contrat`, `salaire`, `congé`, `horaire`, `poste` ;
- administration : `document`, `dossier`, `formulaire`, `demande`, `service`, `administration`, `mairie`, `numéro`, `adresse`, `identité`, `carte`, `preuve`, `signature`, `rendez-vous` ;
- argent : `banque`, `compte`, `paiement`, `facture`, `dette`, `revenu`, `dépense`, `impôt`, `taxe`, `assurance`, `aide` ;
- droit courant : `accord`, `obligation`, `autorisation`, `interdit`, `permis`, `responsabilité`, `plainte`, `juge`, `tribunal`.

### Résultats de validation

- 650 mots dans la liste ;
- 650 entrées dans le dictionnaire ;
- 240 formes dans la table morphologique ;
- 0 mot hors vocabulaire dans les définitions ;
- 0 circularité directe simple ;
- 100 tests d expression travail, administration, argent et droit valides.

### Portée

V3.2 étend V3.1 vers les situations ordinaires de travail, de démarches, de paiement et de droit courant.

### Limites connues

- plusieurs notions administratives et juridiques restent volontairement compressées ;
- la police, l avocat, la procédure, le délai, le chômage et le loyer restent absents ;
- le système vise encore la clarté minimale plus que la finesse institutionnelle complète.
