# Diff V5.1 -> V5.2 sémantique

- Base: `dictionary_v5_1_editorial_proposal.json` et `forms_v5_1_proposal.txt`.
- But: amélioration du sens, en priorité sur les familles encore définies par hyperonyme faible ou placeholder.

## Exemples de réécriture

- `assiette` : `chose de repas` -> `un objet plat pour mettre un aliment ou un repas`
- `goût` : `chose de bouche` -> `un effet d’un aliment ou d’une boisson dans la bouche`
- `camion` : `chose de transport` -> `un grand véhicule pour porter des objets`
- `moteur` : `chose de transport` -> `une partie d’un véhicule ou d’une machine qui produit un mouvement`
- `étoile` : `chose de ciel ou de espace` -> `un point de lumière dans le ciel de nuit`
- `planète` : `chose de ciel ou de espace` -> `un grand corps du ciel qui tourne avec une étoile`
- `atome` : `chose de science` -> `une très petite partie de matière`
- `dessin` : `chose de art` -> `une image faite avec une main ou un outil`
- `rythme` : `chose de musique` -> `un ordre de son ou de mouvement dans le temps`
- `amitié` : `chose de relation` -> `une bonne relation entre des personnes`
- `colère` : `chose de émotion` -> `une émotion forte contre une personne ou contre une chose`
- `pièce` : `chose de maison` -> `une partie d’une maison entre des murs`
- `internet` : `chose de information ou de message` -> `un réseau de connexion entre des ordinateurs pour envoyer, recevoir ou chercher une information`
- `logiciel` : `chose de information ou de message` -> `un ensemble d’ordre pour faire travailler un ordinateur`
- `mot-de-passe` : `chose de information ou de message` -> `un mot privé pour ouvrir un compte ou un service`
- `taureau` : `animal vivant` -> `un grand animal fort, de même espèce qu’une vache`
- `précis` : `mot simple` -> `exact et sans erreur`
- `habitude` : `chose simple` -> `une action souvent faite`
