# Diff V3.1 -> V3.2

- Point de départ: 600 mots.
- Proposition V3.2: 650 mots.
- Mots ajoutés: 50.
- Extension ciblée: travail, administration, argent et droit courant.

## Travail

- `emploi` : `travail de une personne`
- `métier` : `travail de une personne dans un domaine`
- `entreprise` : `groupe de personne pour travail et argent`
- `bureau` : `lieu de travail avec table papier et document`
- `collègue` : `personne qui travailler avec une autre personne`
- `responsable` : `personne qui devoir répondre de un travail ou de un groupe`
- `chef` : `responsable au plus haut de un groupe de travail`
- `client` : `personne qui payer pour un service ou pour une chose`
- `projet` : `travail avec un but et un temps`
- `tâche` : `petite partie de un travail ou de un projet`
- `réunion` : `moment où des personne parler de un travail ou de un projet`
- `contrat` : `accord écrit entre des personne ou des groupe`
- `salaire` : `argent donné pour un travail`
- `congé` : `temps sans travail avec autorisation`
- `horaire` : `heure de début et heure de fin de un travail`
- `poste` : `place de travail de une personne`

## Administration

- `document` : `papier ou page avec information`
- `dossier` : `groupe de document sur une personne ou un projet`
- `formulaire` : `document avec des question et des place pour écrire`
- `demande` : `question ou document pour avoir une chose un droit ou un service`
- `service` : `travail pour aider une personne ou un groupe`
- `administration` : `service public avec document règle et formulaire`
- `mairie` : `lieu de administration de une ville`
- `numéro` : `nom de nombre pour une chose une personne ou un document`
- `adresse` : `nom de lieu de une maison ou de un service`
- `identité` : `information sur le nom de une personne`
- `carte` : `petit document pour montrer une identité ou une autorisation`
- `preuve` : `chose ou document qui montrer que une chose est vrai`
- `signature` : `nom écrit de une personne sur un document`
- `rendez-vous` : `moment que une personne choisir pour voir une autre personne`

## Argent

- `banque` : `service pour argent et compte`
- `compte` : `place de argent dans une banque`
- `paiement` : `action de payer`
- `facture` : `document qui montrer un prix ou un paiement`
- `dette` : `argent que une personne devoir payer`
- `revenu` : `argent que une personne recevoir`
- `dépense` : `argent que une personne payer pour une chose ou un service`
- `impôt` : `argent que une personne ou une entreprise devoir payer à une administration`
- `taxe` : `argent à payer pour une chose ou un service`
- `assurance` : `accord pour protéger contre un risque`
- `aide` : `action ou argent pour aider`

## Droit courant

- `accord` : `oui entre une personne ou un groupe et une autre personne ou un autre groupe`
- `obligation` : `chose que une personne devoir faire`
- `autorisation` : `accord ou document qui permettre une action`
- `interdit` : `pas permis par une règle ou une loi`
- `permis` : `autorisation ou document pour une action`
- `responsabilité` : `obligation de répondre de une action ou de un danger`
- `plainte` : `demande de droit après un mal ou un problème`
- `juge` : `personne de loi qui dire le droit dans un problème`
- `tribunal` : `lieu où un juge voir une plainte ou un problème de droit`

## Effet global

- Le système peut maintenant parler de contrat, salaire, rendez-vous administratif, banque, facture, dette, plainte et tribunal avec des phrases courtes mais utilisables.
- La morphologie a été enrichie seulement pour soutenir ces usages sans toucher à V3.1 stable.
