# README V6.0

## Portée

V6.0 est la version stable conceptuelle du corpus français contrôlé.

- V6.0 part de [wordlist_v5_2.txt](/Users/avialle/dev/minidico/wordlist_v5_2.txt), [dictionary_v5_2.json](/Users/avialle/dev/minidico/dictionary_v5_2.json) et [forms_v5_2.txt](/Users/avialle/dev/minidico/forms_v5_2.txt) ;
- V6.0 ajoute `61` concepts transversaux ;
- V6.0 ne crée pas de nouveau domaine concret ;
- V6.0 étend surtout la capacité du corpus à définir, comparer, argumenter, évaluer et nuancer.

## Nature De L’Extension

Les ajouts portent sur des mots comme :

- `définition`, `explication`, `condition`, `conséquence`, `comparaison`, `précision`, `certitude`, `méthode`, `critère`, `argument`, `durée`, `avis`, `signification`, `logique`, `nuance`.

Le but n’est pas d’élargir les thèmes du monde, mais de mieux parler des relations abstraites entre idées, raisons, règles, jugements, formulations et interprétations.

## Statut Lexical

V6.0 reste lexicalement fermée.

- `1561/1561` définitions conformes ;

- `0` circularité directe simple.

## Tests Conceptuels

V6.0 passe les tests conceptuels avec :

- `300/300` phrases lexicalement valides ;
- `0` phrase impossible ;
- une nette amélioration sur définition, condition, comparaison, probabilité, méthode, évaluation, argumentation et interprétation.

## Position Dans La Série
