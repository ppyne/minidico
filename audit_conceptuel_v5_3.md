# Audit Conceptuel V5.3

## Objet

- Point de départ: V5.2 stable à 1500 mots.
- But: mesurer la capacité du corpus à raisonner, définir, expliquer, comparer et nuancer, sans modifier le lexique.

## Définition et Explication

- Mots présents: expliquer, sens, signifier, texte, message, langue, mot, nom, raison, idée, preuve, montrer, décrire, question, réponse
- Mots manquants: définition, définir, explication, terme, notion, concept, description
- Définitions faibles: signifier -> avoir un sens; texte -> mot avec autre mot
- Tests: naturel 8, acceptable 7, lourd 0, impossible 5
- Candidats V6.0: définition, définir, explication, exemple, notion

## Cause, Conséquence, Condition, Exception

- Mots présents: cause, effet, raison, si, alors, car, mais, ou, donc, possible, impossible, résultat
- Mots manquants: conséquence, condition, exception, sinon, sauf, cas-limite
- Définitions faibles: si -> mot pour une chose possible; car -> mot pour une raison; mais -> mot pour montrer une différence; donc -> mot pour montrer un effet

- Candidats V6.0: condition, conséquence, exception, sauf, sinon
