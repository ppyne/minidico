# README V3

## Objet

`V3.0` est une version stabilisée du dictionnaire français minimal orientée santé, biologie et vie corporelle.

Cette version part de la branche pragmatique `509` mots et l’étend à `550` mots sans remettre en cause les acquisitions précédentes.

## Portée exacte

V3.0 permet de parler de :

- santé courante : symptômes simples, soin, médicament, médecin, urgence ;
- corps et biologie minimale : parties du corps, organe, respiration, muscle, cellule ;
- fonctions corporelles simples : respirer, uriner, saigner, vomir ;
- reproduction et sexualité en vocabulaire neutre : sexe, consentement, grossesse, parent, père, mère, couple ;
- danger et système médical minimal : hôpital, traitement, maladie, symptôme, risque, danger.

## Ce que V3.0 ne cherche pas à faire

- remplacer un vocabulaire médical complet ;
- couvrir la médecine spécialisée ;
- fournir un lexique technique détaillé ;
- couvrir toute la santé reproductive fine.

## Validation

La version stabilisée conserve les propriétés suivantes :

- liste fermée de 550 mots ;
- dictionnaire fermé sur cette liste ;
- table explicite de formes fléchies ;
- validation stricte sans mot hors vocabulaire ;
- 100 tests d’expression santé valides.

## Interprétation

V3.0 doit être compris comme une version utilisable et stable pour parler du corps, de symptômes simples, de soins, de grossesse, de consentement et de danger en français minimal.

Elle n’est pas finale au sens médical fort. Une V3.1 deviendra pertinente si l’objectif est d’ajouter des notions comme `infection`, `virus`, `allergie`, `opération`, `analyse`, `règles` ou `préservatif`.
